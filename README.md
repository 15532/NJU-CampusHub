# CampusHub - 校园互助服务平台

## 📖 项目简介
CampusHub 是一个致力于解决大学校园内碎片化互助需求（如快递代取、二手交易、学习辅导等）的综合服务平台。本平台旨在打破信息壁垒，建立可靠的校园互助信任机制。

## 🛠️ 技术栈全景

**前端架构 (多端复用):**
* **核心框架:** React 18 + Taro (支持一键编译为 H5 网站与微信小程序)
* **UI 生成与设计系统:** Google Stitch (AI 原生 UI 协同生成) + Tailwind CSS

**后端架构 (微服务设计思想):**
* **核心框架:** Java + Spring Boot
* **构建工具:** Maven
* **数据库:** MySQL (采用 MyBatis/MyBatis-Plus 作为 ORM)

**部署与运维:**
* **服务器:** 阿里云 Debian 实例
* **CI/CD:** GitHub Actions 自动化构建与代码审查
